# KaroBooks MVP

Goal: A simple web app for small businesses to create invoices, track GST, and see a dashboard.

## Pages
- Dashboard: revenue, pending, GST collected
- Invoice form: customer, items, amount, GST rate (5/12/18/28%)
- Customer list: all customers with pending vs paid

## Tech (simple)
- Frontend: HTML + CSS + JavaScript (or React if you prefer)
- No backend yet (store data in browser localStorage)
